Initializing the game

We create our sprite resource sets and set the initial game world here.


mygame.cpp


#include "mygame.h"


void gameStart()

{

   setTitle("Amazing game");

   setWindowed(600,400);

   

   SpriteSet *playerSprites=new SpriteSet();

   playerSprites->add("player.png");

   

   new World1();

}


void gameEnd()

{

   exitGame();

}

Functions and methods provided by the engine are in bold text.


void gameStart()

{

   setTitle("Amazing game");

   setWindowed(600,400);

Here we implement our game starting function that we defined earlier. setTitle() will set the name of our window and setWindowed() will create it and set it's size. Making a window should always be thing first thing you do for the engine.


   SpriteSet *playerSprites=new SpriteSet();

   playerSprites->add("player.png");

We define our resource sets here. Making a new set and adding the player.png to it. The first set created will have the index 0 and the first sprite within it will have the index 0. This sprite can now be obtained with getSprite(0,0) or from the set directly if the pointer is stored in some custom data class. Enumerators and data classes are suggested for large projects.


   new World1();

Last thing we do in this function is start up our first scene. Scenes do not need to be destructed as the engine will take care of that. Same for objects.


void gameEnd()

{

   exitGame();

}

Here is our window closing function. exitGame() will take care of ending our game scene, objects and closing the window.
