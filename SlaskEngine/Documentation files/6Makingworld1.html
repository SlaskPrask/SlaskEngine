6. Making world 1

Tutorials ›› Objects and scenes ››

Our first scene

Scenes are essentially game screens or worlds. This scene will be our first world where the game starts in. Normally the first scene is a loading screen scene or the main menu scene.


world1.h


#include "world1.h"


World1::World1()

{

   spriteSet(0)->load();

   

   setSize(1000,400);

   player=at(new Player(),200,200);

   

   cam.setSize(600,400);

   cam.setLimits(width,height);

   cam.setBounds(120,100);

   cam.follow(player);

   cam.setPositionCentered(player);

   cam.activate();

}


World1::~World1()

{

   spriteSet(0)->unload();

}

Functions and methods provided by the engine are in bold text.


   spriteSet(0)->load();

We need to load all the resource sets required by this scene here. Resource sets can also be loaded at the start and the end of the game but we want to manage them per scene. We'll design sprite set 0 to contain our player sprites. Resources should be loaded before any objects are created so they will be able to find the loaded resources.


   setSize(1000,400);

This sets the size of our scene.


   at(new Player(),200,200);

Simply calling new Player() would create us a player but we want him to start at the coordinates 200,200.


   cam.setSize(600,400);

   cam.setLimits(width,height);

   cam.setBounds(120,100);

We construct the camera for our scene here. The limits will determine how far the camera can move and here we restrict it to the size of our scene. The camera's bounds determine how close to our player will be able to get to the edges before the camera starts moving with it.This sets the size of our scene.


   cam.follow(player);

   cam.setPositionCentered(player);

   cam.activate();

The player object will be followed by our camera and we place it so that the player object is int it's center. Finally we activate this camera show the engine will show it in the game window.


World1::~World1()

{

   spriteSet(0)->unload();

}

When the scene unloads we want to unload the resources it has loaded.
