Setting up the game

Now we set up our game files. This is of course up to you how you want it set up.


mygame.h


#include <Slask.h>

using namespace slask;


#include "world1.h"

#include "world2.h"

#include "player.h"


void gameStart();

void gameEnd();

Functions and methods provided by the engine are in bold text.


#include <Slask.h>

using namespace slask;
#include "world1.h"

#include "world2.h"

#include "player.h"

Here we include the engine's header file and header files for the classes we will be designing in this tutorial. world1.h will contain our first scene and world2.h will contain our second scene. Our player object will be in player.h. The namespace contains the engine's built in functions.


void gameStart();

void gameEnd();

We define the start and end point function references here and implement them in mygame.cpp.
