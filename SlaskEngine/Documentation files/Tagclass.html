class Tag


bool runs();

Return whether the tag's objects are allowed to run.

Running is allowed by default.


bool draws();

Return whether the tag's objects are allowed to draw.

Drawing is allowed by default.


Tag* enableRun();

Allows the objects of this tag to run.

Other tags may still prevent the objects from running.

Returns the current tag.


Tag* enableDraw();

Allows the objects of this tag to draw.

Other tags may still prevent the objects from drawing.

Returns the current tag.


Tag* disableRun();

Forbids the objects of this tag from running.

Returns the current tag.


Tag* enableDraw();

Forbids the objects of this tag from drawing.

Returns the current tag.


Tag* disable();

Forbids the objects of this tag from both running and drawing.

This is the same as calling both disableDraw() and disableRun(). Drawing and Running can still be managed separately.

Returns the current tag.


Tag* enable();

Allows the objects of this tag to both run and draw.

This is the same as calling both enableDraw() and enableRun(). Drawing and Running can still be managed separately.

Returns the current tag.
