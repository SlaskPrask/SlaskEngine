Tagging multiple objects together fast


void beginTag(Tag *tag);

All objects created after this function will automatically receive the given tag.

Same as calling addTag(tag) individually on each object.

After creating the objects endTag(tag) should be called.


Generated errors:

Function beginTag() called twice on the same tag.

beginTag(tag) was already called on this tag.


void endTag(Tag *tag);

Stop tagging objects with the tag previously given with beginTag(tag).


Generated errors:

Function endTag() called on a tag that is not active.

beginTag(tag) was not called for this tag before or endTag(tag) was already called once on it.



using namespace slask;


Tag *pauseMenuTag=new Tag();        //create a tag
beginTag(pauseMenuTag);             //begin tagging all future objects with pauseMenuTag
   new playerObject();             //creating an object, it will receive pauseMenuTag as a tag

   new enemyObject();              //creating an object, it will receive pauseMenuTag as a tag

   new enemyObject();              //creating an object, it will receive pauseMenuTag as a tag
endTag(pauseMenuTag);               //end tagging all future objects with pauseMenuTag


Creating an object at a given position


Object* at(Object *obj,double x,double y);

Creates the object at the given location.

Returns the created object.

Returns the same type as the first parameter which can be any child class of Object.



Player *player=at(new Player(),300,200);


Destroying all objects


void destroyAllObjects();

Same as calling destroy() on all objects individually.

Also destroys all persistent objects.
