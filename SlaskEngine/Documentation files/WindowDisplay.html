Changing window size and settings

One of setWindowed(w,h), setFullscreen(w,h) or setFullscreenWindow(w,h) should be called when the game starts.


bool setWindowed(int w,int h);

Changes window size to given width and height.

Makes the game windowed.


Generated errors:

Unable change to window x,y

A fatal error occurred when creating or changing the window.


bool setFullscreen(int w,int h);

Changes window size to given width and height.

Makes the game fullscreen.

This function should only use resolutions provided by the resolution functions described below.


Generated errors:

Unable change to fullscreen resolution x,y

A fatal error occurred when creating or changing the window.

Attempted to switch to unsupported resolution x,y

Width and height given are not supported by the current display.

This error is not fatal and will keep the original window.


bool setWindowedFullscreen(int w,int h);

Changes window size to given width and height.

Makes the game a borderless fullscreen window.


Generated errors:

Unable change to fullscreen windowed x,y

A fatal error occurred when creating or changing the window.


int getWindowWidth();

Returns the current width of the window.


int getWindowHeight();

Returns the current height of the window.


void setTitle(const char* str);

void setTitle(std::string str);

Set's the text shown in task bar and at the top of the window.


void setFPS(int fps);

Sets the current frames per second value of the game for both drawing and running.

Should be called when the game starts.

60 by default.


void enableVSync();

Enables vertical syncing of the game window.

Enabled by default.


void disableVSync();

Disables vertical syncing of the game window.

Enabled by default.


const char* getTitle();

Returns the title of the game window.


int getFPS();

Returns the game's current frames per second value.


bool getVSync();

Returns whether vertical syncing is enabled for the game window.



using namespace slask;


setTitle("Hello World!");

setWindowed(640,480);

enableVSync();

setFPS(60);


Changing window scaling options

Window scaling when using a camera instance can be changed. This scaling does not apply when there is no camera, the game window will just show as much as it can. Custom scaling can also be done by settings a custom resize function with setGameWindowResizeFunction and adjusting the active camera there. This way you can create nicer context outside the game region instead of having black borders.


void windowScalingFit();

Stretches the current camera to the window's size maintaining it's aspect ratio and adding black bars to the sides of the window.

This is the default scaling option.


void windowScalingStretch();

Stretches the current camera to fit the entire window.


Obtaining supported resolutions


int getResolutions();

Returns the number of supported resolutions.


int getResolutionWidth(int i);

Returns the width of the resolution at index i.


Generated errors:

Attempting to read supported resolution width i/x out of range.

A fatal error occurred when creating or changing the window.


int getResolutionHeight(int i);

Returns the height of the resolution at index i.


Generated errors:

Attempting to read supported resolution height i/x out of range.

A fatal error occurred when creating or changing the window.



using namespace slask;


int largestWidth,largestHeight;

int lastResolution=getResolutions()-1;              //index to the last supported resolution

largestWidth=getResolutionWidth(lastResolution);    //get the width of the last resolution

largestHeight=getResolutionHeight(lastResolution);  //get the height of the last resolution

setFullscreen(largestWidth,largestHeight);          //set the game to fullscreen mode
