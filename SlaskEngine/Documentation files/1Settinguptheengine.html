Setting up the engine

Similar to the Hello World tutorial we set up the engine.


main.cpp


#include <Slask.h>

#include "mygame.h"


using namespace slask;


void main(int argc,char *argv[])

{

   setGameStartFunction(&gameStart);

   setGameEndFunction(&gameEnd);

   

   delete new SlaskEngine(argc,argv);

   

   return 0;

}

Functions and methods provided by the engine are in bold text.


#include "mygame.h"

We include a header file we will be writing in the process of this tutorial that includes all our game header files.


   setGameStartFunction(&gameStart);

   setGameEndFunction(&gameEnd);

Here we set the game's start and end event functions. We will be defining these in mygame.h and mygame.cpp.
