Game entry and exit points

Optional functions that can be set to trigger on certain engine events.


void setGameStartFunction(void(*func)(void));

Sets the given function pointer to be called when the game starts.

This function should set up the initial settings of your game including creating a window.

Use this function instead of settings them up in main().


Generated errors:

No game entry point given.

No function was given with setGameStartFunction() and the program will close.


void setGameEndFunction(void(*func)(void));

Sets the given function pointer to be called when the player is attempting to close the game window.

In this function you can set up asking the player if he's sure he wants to quit or if he wants to save.

If this function is not defined then the program will simply close.


void setGameWindowResizeFunction(void(*func)(void));

Sets the given function pointer to be called when the game window is resized.

New size can be obtained from getWindowWidth() and getWindowHeight().



#include <Slask.h>


using namespace slask;


void gameStart()

{

   setWindow(640,480);                 //set the initial game window

}


void gameEnd()

{

   exitGame();                         //close the program

}


void windowResized()

{

   int windowWidth=getWindowWidth();   //obtain the new width

   int windowHeight=getWindowHeight(); //obtain the new height

}


void main(int argc,char *argv[])

{

   setGameStartFunction(&gameStart);

   setGameStartFunction(&gameEnd);

   setGameStartFunction(&windowResized);

   

   delete new SlaskEngine(argc,argv);

   

   return 0;

}
