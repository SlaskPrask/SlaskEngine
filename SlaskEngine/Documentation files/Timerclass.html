class Timer

The timer class can be used to measure passing time in milliseconds. These timers run even when object tags are paused and are independent from the game flow.


void getTime();

Returns the measured time in milliseconds.


void isRunning();

Returns whether the timer is not paused or stopped.


void start();

Starts the timer. Initial time passed is set to 0ms.


Generated errors:

Timer is already running.

        start() was already called on this timer.


void stop();

Stops the timer.


void restart();

Resets a running timer's passed time to 0 and starts it.

If the timer is stopped it will call start() on it.


void pause();

Pauses the current timer if it is running.

Does nothing to timers that aren't running.


void unpause();

Resumes paused timers.

Calls start() on timers that aren't running.



void player::player()

{

   timer.start();             // start the timer

}


void player::run()

{

   if (timer.getTime()>1000) // check if one second has passed

   {

       x+=10;                 // move x every second

       timer.restart();       // restart the timer

   }

}
