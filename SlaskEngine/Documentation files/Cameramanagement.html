Handling the active camera

Camera position and size can still be obtained when there is no active camera. This will essentially return the top left are of the game currently shown in the game window.


Camera* getCamera();

Returns the active camera.

Returns NULL when no camera is active.


void deactivateCamera();

Releases the currently active camera. Does not delete it.


void GetCameraX();

Returns the current camera's x position.


void GetCameraY();

Returns the current camera's y position.


void GetCameraWidth();

Returns the current camera's width.


void GetCameraHeight();

Returns the current camera's height.


Creating a camera at a given position


void at(Camera *cam,double x,double y);

Returns the current camera's height.



using namespace slask;


Camera *cam=at(new Camera(),300,200);

cam->setSize(640,480);

cam->activate();
