Mouse button names


Getting the name of a button

void getMouseName(char *str,int i);

void getMouseName(std::string *str,int i);

Inserts the name of the given mouse button in i to str.

If you wish to support other languages or implement your own key names you may write your own function.


Generated errors:

Reading mouse button i name out of range.

The given button, i, is not a recognized mouse button.



using namespace slask;


int fireButton=Mouse::MouseLeft;                //assign a button for firing


char str[50];

drawText(font,"Fire:",0,0,16,1,1,1,1);
