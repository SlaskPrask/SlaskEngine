class Camera

Cameras are used to crop out a part of the scene. It can be smaller than the scene to provide a zoom effect on the focused area or the same size as the scene. Defining an active camera makes it's view scale to the window. When no camera is active the window will display as much as it can in native scale. Inactive cameras can still be used for positioning purposes.


Provided members and methods


Public members

double x;

The camera's x position.


double y;

The camera's y position.


Creating a camera


Camera();

Camera(double xp,double yp,double sw,double sh);

Creates a new camera. Initial top left position can be given in xp,yp and initial width/height size in sw,sh.


Activating the camera


void activate();

Makes the window show the view of this camera.

Deactivates all previous cameras.

If you wish to have no camera active you may call slask::deactivateCamera().


void isActive();

Returns whether the camera is currently active.


Moving the camera

x and y values can be directly changed and read.


void at(double xp,double up);

void setPosition(double xp,double up);

Changes the position of the camera to xp,yp.


void setPositionCentered(double xp,double up);

Changes the position of the camera to center on xp,yp.

Same as calling setPosition(xp-getWidth()/2,yp-getHeight()/2).


void setPositionCentered(Object *obj);

Changes the position of the camera to center on obj's x and y.

Same as calling setPosition(obj->x-getWidth()/2,obj->y-getHeight()/2).


double getLeft();

Returns the left edge of the camera. Same as x.


double getTop();

Returns the top edge of the camera. Same as y.


double getRight();

Returns the right edge of the camera. Same as x+getWidth().


double getBottom();

Returns the bottom edge of the camera. Same as y+getHeight().


Resizing the camera


void setSize(double sw,double sh);

Changes the width of the camera to sw and the height to sh.


double getWidth();

Returns the width of the camera.


double getHeight();

Returns the height of the camera.


Making the camera automatically follow a point


void follow(double *xpoint,double *ypoint);

Makes the camera automatically follow the given coordinate pointers.


void follow(Object *obj);

Makes the camera automatically follow the given object's x and y coordinates.

Same as calling follow(&(obj->x),&(obj->y)).


void unfollow();

Stops the camera from automatically following the given coordinate pointers.


bool isFollowing();

Returns whether the camera is actively following a set of coordinate pointers.


void setBounds(double left,double top,double right,double bottom);

Defines close to the edges of the camera it will let the followed point get.

When the bounds are too large for the camera it will center itself on the camera.


void setBounds(double hor,double ver);

Calls setBounds(hor,ver,hor,ver) on the camera.

Fast way to set symmetrical boundaries.


void setLimits(double left,double top,double right,double bottom);

Limits the camera's boundaries from going past the given limits.

This does not prevent the followed point from going past the limits and the point can reach the edges of the camera when the camera's position is at it's limits.

Usually the limits are set to match the size of the current scene.


void setLimits(double right,double bottom);

Calls setLimits(0,0,right,bottom) on the camera.

Fast way to set limits that start from 0,0 position.


void unsetLimits();

Removes the limits set to the camera.


double getBoundsLeft();

Returns the left boundary of the camera.


double getBoundsTop();

Returns the top boundary of the camera.


double getBoundsRight();

Returns the right boundary of the camera.


double getBoundsBottom();

Returns the bottom boundary of the camera.


double getLimitsLeft();

Returns the left limit of the camera.


double getLimitsTop();

Returns the top limit of the camera.


double getLimitsRight();

Returns the right limit of the camera.


double getLimitsBottom();

Returns the bottom limit of the camera.



cam=new Camera();

cam->size(640,480);

cam->setPositionCentered(player.x,player.y);

cam->setBoundaries(140,100);

cam->setLimits(0,0,getSceneWidth(),getSceneHeight())

cam->follow(player);

cam->activate();
