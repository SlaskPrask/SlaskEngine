First example

This tutorial will help you set up the first running instance of the game.


main.cpp


#include <Slask.h>


using namespace slask;


void gameStart()

{

   setWindow(640,480);

}


void gameEnd()

{

   exitGame();

}


void main(int argc,char *argv[])

{

   setGameStartFunction(&gameStart);

   setGameEndFunction(&gameEnd);

   

   delete new SlaskEngine(argc,argv);

   

   return 0;

}

Functions and methods provided by the engine are in bold text.


#include <Slask.h>

This header file includes all that is needed to make the engine run.


using namespace slask;

The namespace contains all the functions to handle the engine's classes and functions.


void gameStart()

{

   setWindow(640,480);

}


void gameEnd()

{

   exitGame();

}

We define the custom gameStart() function that we want to use for initializing our game. This function can contain loading your settings, resources, loading, and settings up the first scene of your game. For now we only create the game window with it. This function should always create a window. setWindow(640,480) is used to create our game window.


gameEnd() function is defined as the exit point of your game. When you attempt to close the game window this function will be called. Here we can ask the player if he's sure he wants to exit and whether he would like to save or not. This function does not have to be the end of the game but should start a fast sequence to ending it. exitGame() is used to instantly close the engine and the program. If you wish to have the player do actions when the game is being closed don't call exitGame() instantly.


void main(int argc,char *argv[])

{

   setGameStartFunction(&gameStart);

   setGameEndFunction(&gameEnd);

Here is the main function of our game. We give the pointer to our gameStart function using the engine's setGameStartFunction() function. We also give the pointer to the endGame function with setGameEndFunction().


   delete new SlaskEngine(argc,argv);

This creates the engine and runs it until the game flow ends and it is all that is needed to start the game.


   return 0;

}

Last we cleanly finish our program.


When compiled and ran you should see a black window titled "Game" appear.
